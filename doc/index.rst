Believe-It-or-Not
=================

Believe-It-or-Not — многопользовательская терминальная карточная
игра «Верю — не верю», написанная на Python 3.12.

Архитектура проекта
-------------------

Проект построен по клиент-серверной архитектуре.

``believe.client``
    Терминальный клиент на основе ``cmd.Cmd``. Принимает команды
    пользователя, отправляет их серверу и выводит сообщения.

``believe.server``
    Асинхронный сервер на основе ``asyncio.start_server``.
    Регистрирует пользователей, формирует партии и обрабатывает
    сетевое взаимодействие.

``believe.server.game``
    Игровая логика. Содержит классы карты, колоды, игрока,
    последнего хода и партии.

``believe.common``
    Общие константы клиента, сервера и игровой модели.

Взаимодействие компонентов
--------------------------

Клиент передаёт серверу текстовые команды, каждая из которых
завершается переводом строки.

Сервер самостоятельно обрабатывает команды ``start``, ``yes``,
``locale`` и ``quit``.

Команды ``play``, ``believe``, ``not`` и ``rules`` передаются
методу ``Game.process()``.

Игровая модель возвращает личный ответ и список событий, которые
сервер последовательно обрабатывает и рассылает клиентам.
