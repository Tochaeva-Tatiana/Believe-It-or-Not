# Believe-It-or-Not

Командный проект по игре «Верю не верю» на Python 3.12.

---

## Описание проекта

Приложение реализует карточную игру «Верю — не верю».

Первый игрок кладёт на стол одну или несколько карт, объявляет их количество и называет достоинство карт. Следующий игрок должен решить, верит он этому заявлению или нет.

Если игрок не верит, он проверяет одну из выложенных карт, выложенных предыдущим игроком. Если карта действительно соответствует объявленному достоинству, проверяющий забирает всю стопку себе. Если же первый игрок соврал, то он забирает стопку.

Если игрок верит, он добавляет в стопку любое количество своих карт и объявляет, что они имеют то же достоинство. Затем следующий игрок также выбирает: поверить или проверить. Игра продолжается по такому же принципу

Проект поддерживает локальную игру.
---

## Основные возможности

- Ввод карт игрока
- Выбор ответа: «верю» / «не верю»

- Поддержка нескольких игроков
- Локальный режим
- Локализация интерфейса
